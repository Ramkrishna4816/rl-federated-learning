# RL-Based Federated Learning Controller

This project explores Reinforcement Learning for improving Federated Learning under IID and Non-IID client data settings.

## Features
- Federated Learning simulation
- Non-IID data distribution
- Client drift analysis
- RL-based decision controller
- Accuracy/loss comparison graphs

## Tech Stack
Python, PyTorch, NumPy, Matplotlib, Google Colab

## How to Run
Open `notebooks/RL_Federated_Learning.ipynb` in Google Colab and run all cells.
